Custom the icon in Wechat share link.

== Description ==
The plugin allow you custom the icon when you share link in Wechat to your friend or timeline. Use Wechat API, custom description and other meta infomation will be support in future version.

== Installation ==
Download
Upload to your /wp-contents/plugins/ directory.
Activate the plugin through the \'Plugins\' menu in WordPress.
Go to Setting -> Wechat Share, then check the post type you want to custom the icon and save setting. Post, Page and Attachment are default checked post type

== Frequently Asked Questions ==
Why I can\'t find the meta box in other post type?
Go to Setting > Wechat Share, then check the post type you want to custom the icon and save setting.
